# ✨ Mishi AI

Mishi AI is a modern AI chatbot powered by Google's Gemini API.

## Features
- Modern AI assistant interface
- Responsive design
- Chat history sidebar
- Suggested prompts
- Typing indicator
- Gemini AI integration

## Project Structure

```text
Mishi-AI/
├── mishi-ai.html
├── ChatBot.py
├── requirements.txt
├── LICENSE
├── .gitignore
└── README.md
```

## Installation

```bash
pip install -r requirements.txt
python ChatBot.py
```
